# SOC Lab

Servidor de monitorizacion para clientes de Linux/Windows.

## Herramientas utilizadas
* Wazuh
* Sysmon

## Recursos
### Windows Client
* Sistema Operativo: Windows 10 Pro
* RAM: 4GB
* Nucleos de CPU: 2
* Disco: 40GB
### Wazuh Server
* Sistema Operativo: Linux Ubuntu Server 24.04.3
* RAM: 4GB
* Nucleos de CPU: 2
* Disco: 40GB

## Metodo de instalacion de Wazuh
Modo All-in-One: Manager + Indexer + Dashboard en un unico servidor.
